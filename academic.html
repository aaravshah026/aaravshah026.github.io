Research

Researcher with GT's SIPLab

I programmed a script to emulate local field potentials in simulations of the human brain in the presence of a photon. Data is given in a CSV file, then the script extrapolates to a 3D graph of amplitude values to multiply to field potentials. I also built a Python package to allow real-time alternatives to a Hilbert transform function. A phase locking approach, nonresonant linear filter, and resonant linear filter were included. I then created a Jupyter notebook to demonstrate the accuracy of example tests, including those published to Scientific Reports.
